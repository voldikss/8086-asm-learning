# 第六章 包含多个段的程序

- `dw`：定义字型数据
- `db` 定义字节型数据(ch07)
- `dd`：定义双字型数据(ch08)

- `end`

  - 通知编译器程序结束
  - 告知编译器程序的入口

- 分段

  - 数据、栈、代码分段
  - 如果所需要的空间超过 64KB，就不能放在同一个段中，因为 8086 中一个段的容量不能大于 64KB
  - 段名相当于一个标号，代表了段地址

- `assume` 并不会主动将 cs,ds,ss 分别和我们定义的 code,data,stack 相连，因为它是伪指令，是由编译器执行的而非 CPU 执行。CPU 对我们定义于段中的内容当做数据、指令、还是栈空间，仅仅由 CS:IP, SS:SP,DS 等寄存器的设置来决定
